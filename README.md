# iTerm 2 - Color Themes

A collection of my personal themes for [iTerm2](https://iterm2.com/)


## Import Steps

```
Profiles >> Open Profiles ... >> 
Edit Profiles >> Colors >>
Color Presets ... >> Import...
```
